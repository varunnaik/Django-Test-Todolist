API Reference

All API calls use HTTP Basic Authentication.
List all todo items
GET http://localhost:8000/api/v1/todo/
Create a new todo item
POST http://localhost:8000/api/v1/todo/
with a new item as JSON. eg. {'name': 'My task', 'user': 'http://localhost:8000/todo/api/user/1', 'due': '2012-12-20'}
The 'name' and 'user' fields are mandatory, everything else is optional.
Modify a todo item
PUT http://localhost:8000/api/v1/todo/{your_item_id}
with a JSON object containing new values for the item. Fields not specified will retain their original values
Delete a todo item
DELETE http://localhost:8000/api/v1/todo/{your_item_id}
to delete the item specified by {your_item_id}
